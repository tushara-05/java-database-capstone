adminDashboard.html (Thyemeleaf Code) In this section, you’ll build the main dashboard for Admin users. This page will serve as the control panel for managing doctors — allowing admins to view, add, delete, and filter doctor profiles. The page should include a search bar, dropdown filters (by specialty or time), and a dynamic list of doctor cards. You'll also add a modal popup for adding new doctors, which will be triggered by a button click. This layout will eventually be connected to your backend and populated using JavaScript services. Tips: The admin is validated first using token and userRole. Include modal forms for adding new doctors.
